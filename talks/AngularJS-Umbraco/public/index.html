Package Manifest

Defining our property editor

"propertyEditors": [
  {
    "alias": "GoogleMaps",
    "name": "Google Map Location Picker",
    "editor": {
      "view": "~/app_plugins/googlemaps/editor.html",
      "valueType": "JSON"
  }
]

Package Manifest

Adding in configuration options

"prevalues": {
  "fields": [{
    "label": "Initial Latitude",
    "description": "This is used when no value set",
    "key": "initialLat",
    "view": "string"
  }]
}
