What is Analytics for Umbraco?

  • Easily view & monitor your Google Analytics reports
  • An Open Source project to help learn Angular for Umbraco

Animate gif or HTML5 video of it in action

Why did I build this?

  • The outstanding version by Morten Christensen no longer works due to API changes by Google
  • To learn how to extend Umbraco 7 with Angular & the new Umbraco APIs available in the core
  • To help teach myself new development techniques with a 'real' project with an ultimate project deliverable

Adding a Custom Section

//Get SectionService
var sectionService = applicationContext.Services.SectionService;

//Try & find a section with the alias of "analytics"
var section = sectionService.GetByAlias("analytics");

if (section == null) {
  //It doesn't exist, so let's create the section
  //Friendly Name, Application Alias, Icon
  sectionService.MakeNew("Analytics", "analytics", "icon-pulse");
}
